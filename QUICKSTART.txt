ğŸ› ï¸  TROUBLESHOOTING
====================

Error: "Module 'nautilus_trader' not found"
-------------------------------------------
Solution: pip install -U "nautilus_trader[docker,ib]" --break-system-packages

Error: "Redis connection failed"
---------------------------------
Solution: docker run -d -p 6379:6379 redis:latest

Error: "Permission denied on setup.sh"
--------------------------------------
Solution: chmod +x setup.sh

Error: "No results generated"
-----------------------------
Solution: Check that logs/ directory exists (mkdir -p logs)
